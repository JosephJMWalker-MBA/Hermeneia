# Architecture Decision Records

ADRs are case law beneath the Constitution, Authority Index, ratified
amendments, and constitutional invariants. They remain immutable historical
records.

An ADR may be wholly or partially superseded only through the append-only
register in [`../01_Authority_Index.md`](../01_Authority_Index.md). A
supersession notice does not erase the original decision.

ADRs capture permanent architectural decisions and their rationale.

Format: - Status - Context - Decision - Consequences - Alternatives
Considered
