# Ratified Constitutional Amendments

This directory contains append-only amendments to the Constitution of
Hermeneia.

An amendment is authoritative only when its status is `RATIFIED` and it is
listed as `ACTIVE` in [`../01_Authority_Index.md`](../01_Authority_Index.md).

Amendments shall not silently edit or erase earlier constitutional text.
Supersession is recorded in the Authority Index.
